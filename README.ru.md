✨ Плагин CustomEnchantments
Плагин для серверов Minecraft (Paper/Spigot), расширяющий стандартную систему зачарования,
добавляя уникальные «механики» для инструментов, оружия и брони, а также систему археологии с артефактами.

🚀 Особенности
Наковальня: Кастомные зачарования можно объединять и переносить на предметы через AnvilListener.
Редкость и баланс: Каждое зачарование имеет свою степень редкости и уровни прокачки.

🛠 Техническая информация
Плагин использует систему NamespacedKey для регистрации зачарований, что гарантирует совместимость с другими плагинами.

⚙️ Установка
1) Скачайте собранный .jar файл.
2) Поместите его в папку /plugins/ вашего сервера.
3) Перезапустите сервер.
4) Готово! Новые зачарования начнут появляться в игре.
