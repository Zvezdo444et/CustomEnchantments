✨ CustomEnchantments Plugin
A plugin for Minecraft servers (Paper/Spigot, version 1.16.5) that expands the standard enchantment system 
by adding unique "mechanics" for tools, weapons, and armor, as well as an archeology system with artifacts.

🚀 Features
Anvil: Custom enchantments can be combined and transferred to items via AnvilListener.
Rarity and Balance: Each enchantment has its own rarity and upgrade levels.

🛠 Technical Information
The plugin uses the NamespacedKey system to register enchantments, ensuring compatibility with other plugins.

⚙️ Installation
1) Download the compiled .jar file.
2) Place it in the /plugins/ folder of your server.
3) Restart the server.
4) Done! New enchantments will begin appearing in the game.
